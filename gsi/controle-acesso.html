Controle de Acesso

Gerencie separadamente os usuários administrativos e os usuários vinculados a contrato. Responsáveis cadastrados em contratos ficam apenas como referência de assinatura.

Usuários administrativos

Cadastrar usuário administrativo Inclui usuário administrativo com perfil de acesso, login, senha temporária e contato. Tabela de usuários administrativos Consulta os usuários administrativos cadastrados para acesso ao sistema administrativo. Remover usuário administrativo Retira o usuário administrativo da lista ativa e registra o motivo da remoção.

Usuários vinculados a contrato

Cadastrar usuário do contrato Inclui usuário autorizado a gerar vagas para um contrato específico. Tabela de usuários do contrato Consulta os usuários vinculados a contrato cadastrados pelo GSI. Remover usuário do contrato Retira o usuário vinculado a contrato da lista ativa e registra o motivo.
